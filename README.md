# Akan Name Generator

## Project Description
The Akan Name Generator is a simple web application that calculates the day of the week a user was born based on their birth date and assigns them an Akan name according to their gender. Akan names originate from Ghanaian culture, where children are traditionally named after the day of the week they were born.

This project demonstrates the use of HTML forms, CSS styling, and JavaScript logic to handle user input, validation, calculations, and dynamic content display.

## BDD (Behavior Driven Development)

- **Given** the user enters a valid date and selects a gender  
- **When** the user submits the form  
- **Then** the application calculates the day of the week correctly  
- **And** displays the correct Akan name  

- **Given** the user enters an invalid date or leaves fields empty  
- **When** the form is submitted  
- **Then** the application displays an alert message
